# ToDoApp

To-Do app made during hybrid mobileprogramming-course.

## Functionality

With this app, users can (at the moment):

- Add a new to-do item to a list
- View a list of things to-do

## Technologies

This app has been built using:

- React Native CLI
- Metro build tool/dev server
- Android emulator

## What I learned?

During this project, I learned how React Native works and how to build CRUD-apps with it.
I already knew React, so I was familiar with the basic concepts of React Native.
